2. 네트워크 용어
   1) IP 주소 :  네트워크상에 유일하게 식별될 수 있는 네트워크 장치의 주소이다.
                  4개 숫자가 점(.)으로 연결되어 있으며, 각각은 0~255로 표현이 가능 (예) 192.156.11.15
                  현재 4개의 숫자로 표현된 IP주소는 32bit IPV4로 사용된다. 고갈되면 128bit IPV6으로 사용될 추세
                  도메인 네임, 네트워크 환경에 따라 (고정IP, 유동IP)

   2) 내 컴퓨터 IP주소 확인은 ipconfig

   3) 포트(PORT) : 네트워크를 통해 데이터를 주고 받는 출입구, 0~65535
                       포트를 열어놓고 연결요청이 오기를 기다리는것은 서버
                       그 포트로 연결요청을 보내는 것은 클라이언트
  
   4) 도메인 네임 : www.naver.com
   
   5) DNS 서버 : www.naver.com은 DNS서버를 통해  해당 네이버 ip 반환 202.179.177.21
   
   6) URL (Uniform Resource Loaction) 인터넷 상의 자원(리소스)에 대한 주소를 말한다.
       http(프로토콜 식별자) :(구분자)  //www.naver.com/news(리소스)

   7) www.naver.com(호스트 이름) /index.html(파일이름) :8080(포트번호)
   
 3. 네트워크 통신 종류
   1) TCP/IP transfer control protocol/Internet protocol  : 전화기 통신
      IP프로토콜 : 네트워크에 연결된 개개의 컴퓨터에 유일한 주소
      TCP프로토콜 : IP 프로토콜 위에서 연결형 서비스를 지원하는 전송계층 프로토콜
      
   2) UDP  user datagram protocol   : 소포방식, 편지
      비연결 지향적 프로토콜이다. 비연결 지향적이란 데이터를 주고 받을 때 연결 절차를 거치치 않고,
      발신자가 일방적으로 데이터를 발신하는 방식이다.
